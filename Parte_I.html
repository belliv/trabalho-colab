Objetivo

Definição do tipo de problema



O problema trabalhado neste dataset é de regressão, pois o objetivo é prever um valor numérico contínuo, correspondente à pressão sonora ("scaled-sound-pressure"). Diferentemente de um problema de classificação, não se deseja prever uma categoria, mas sim um valor real.
 

Treinamento do modelo



Como o objetivo é prever um valor numérico (pressão sonora), este problema é classificado como um problema de regressão.

Para o treinamento será utilizado o algoritmo Random Forest Regressor, pertencente à biblioteca Scikit-Learn. Esse algoritmo é bastante utilizado por apresentar boa capacidade de previsão e reduzir problemas de sobreajuste (overfitting).
